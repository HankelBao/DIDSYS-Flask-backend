## APIs
+ scores

    - get ranking of classes based on time period
+ inspectors
    - score
    - change previous scores
+ admin
    - add subject
    - del subject
    - update subject full scores
    - add class
    - del class
    - add inspector
    - del inspector
    - update inspector subjects 
    - update inspector classes
